# 1️⃣ MARKOV MODELS / STATE TRANSITION MATRICES

## Core idea:

A system moves between states with certain probabilities that depend **only on the current state**.

---

## Example states:

* Range (R)
* Trend (T)
* Breakout (B)
* Crash (C)

---

## Transition matrix:

```
        R     T     B     C
R     0.6   0.2   0.15  0.05
T     0.1   0.7   0.1   0.1
B     0.0   0.5   0.4   0.1
C     0.2   0.3   0.0   0.5
```

---

## Interpretation:

* Each row = current state
* Each column = next state
* Values = probability of transition

---

## Key property:

> The system is **memoryless** (Markov property)

Meaning:

* Future depends only on current state
* Not on full history

---

## Insight:

> Charts can be seen as trajectories through a state transition matrix.

---

# 3️⃣ BAYESIAN UPDATING (REAL-TIME PROBABILITY ADJUSTMENT)

New data:

After updating:

* 80% chance of breakout

## Insight:

> Beliefs evolve continuously as new signals arrive.
